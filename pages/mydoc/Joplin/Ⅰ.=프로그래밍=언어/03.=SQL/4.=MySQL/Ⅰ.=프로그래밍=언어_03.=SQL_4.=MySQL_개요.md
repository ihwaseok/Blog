## MySQL
- 오라클에서 관리 및 지원하고 있는 오픈소스 관계 데이터베이스 관리 시스템
- 다양한 운영체제 지원
<br>

## 특징
- 오픈소스 라이센스
- 다양한 프로그래밍 언어 지원
- 대용량 데이터를 효과적으로 처리 가능
- 표준 SQL 형식 사용
- MySQL 프로그램을 사용자의 용도에 맞게 수정 가능
